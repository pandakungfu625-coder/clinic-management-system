# Clinic Management System

This repository is a converted student project adapted into a minimal **Clinic Management System**.

Features added:
- Patients (CRUD)
- Doctors (CRUD)
- Appointments (schedule, update, delete, list)

How to run locally:

```bash
python app.py
```

Open http://localhost:8000 and use the SPA pages:
- /patients
- /doctors
- /appointments

Tests:

```bash
python -m unittest -v
```
