Student Management System

Full-stack mini project (Students • Courses • Enrollments • Profiles)

This project demonstrates how a real web app works end-to-end: a frontend UI talking to a backend API, and data stored in a relational database. It covers CRUD, relationships, and JOIN outputs in a simple, explainable way.

Frontend: HTML + Tailwind + JS Backend: Python HTTP Server Database: SQLite Concepts: CRUD + JOIN
