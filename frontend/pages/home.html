Clinic Management System

Manage patients, doctors and appointments with a simple, single-page UI.

This clinic demo demonstrates a lightweight full-stack app: a frontend SPA talking to a Python backend and a SQLite database. It includes CRUD operations and a simple appointments report.

Frontend: HTML + Tailwind + JS Backend: Python HTTP Server Database: SQLite Domain: Clinic MVP
